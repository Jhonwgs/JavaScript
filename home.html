Oque é uma variavel?
Variavel é um espaço na memória que é responsável por armazenar determinadas informações, essas variaveis possuem tipos. em JS usamos let para definir a criação de uma variavel e tudo deve ser finalizado com ; Nomes de variaveis não podem começar com número e não deve haver espaço entre o nome de uma variável em JS os nomes das variaveis são cameSensitive ou seja maiusculo e minusculo fazem diferença indica-se utilizar o camelCase que é uma letra maiuscula e uma minuscula para diferenciar as palavras
